Instalasi Aplikasi

Aplikasi ini memerlukan database mandiri agar bisa digunakan secara permanen oleh instansi Anda.

PENTING: Pastikan Anda telah mengatur Rules Firestore ke "Test Mode" sementara agar pembuatan Akun Admin di Langkah 3 berhasil.

1. Konfigurasi Firebase (Database Utama)

Salin seluruh blok firebaseConfig dari Firebase Console dan tempel di kotak bawah ini.

2. Google Gemini API (Engine AI)

3. Setup Akun Administrator

Buat kredensial login utama Anda. Akun ini akan otomatis disimpan ke database Anda yang baru.
